Request a new git repository

Here you can request a new git repository. To request the main repository, $project.git, simply leave the 'Repository Name' field blank. The PMC selection will fill in the $project field.
MAKE SURE THE NOTIFICATION LIST AND REPO NAME ARE BOTH CORRECT!.
Incubator podling repos should be named as incubator-$podling[-$subname].git

If you wish the repo to have write access to ALL Apache committers, aka "Universal Commit Bit," please open an INFRA Jira ticket after creating the repo.

You MUST be on the PMC or PPMC of the project you wish to create a new repository for.
